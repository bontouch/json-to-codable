JSON to Codable Swift Object

Paste JSON in the text area below and click on GENERATE to generate corresponding Swift objects.

Restrictions:

  • Top level object must not be an array.
  • Arrays will be treated as uniform arrays that only contain one type.
  • Only the first element of the array will be considered.
  • Arrays with array elements (e.g. [[String]]) are not yet supported.
  • All properties that are not arrays are currently optional.
  • Expect bugs. :-)
